## 🚀 How to Use

### Quick Start
```bash
# Natural language commands
./bin/ask-nix "install firefox"
./bin/ask-nix "web server with nginx"
./bin/ask-nix "search for markdown editor"

# Interactive mode
./bin/ask-nix --interactive

# Beautiful TUI (requires textual)
./bin/nix-tui

# REST API (optional)
python run-api.py
```

### Demo
