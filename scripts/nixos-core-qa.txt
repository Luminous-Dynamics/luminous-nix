How do I install Firefox on NixOS?|To install Firefox on NixOS, you have several options:\n\n1. **System-wide (recommended)**: Edit /etc/nixos/configuration.nix and add firefox to environment.systemPackages:\n```nix\nenvironment.systemPackages = with pkgs; [\n  firefox\n  # other packages...\n];\n```\nThen run: `sudo nixos-rebuild switch`\n\n2. **For current user only**: `nix-env -iA nixos.firefox`\n\n3. **Temporarily**: `nix-shell -p firefox`
How do I update my NixOS system?|To update NixOS:\n\n1. **Update channels**: `sudo nix-channel --update`\n2. **Rebuild system**: `sudo nixos-rebuild switch`\n\nOr do both at once: `sudo nixos-rebuild switch --upgrade`\n\nTo preview changes without applying: `sudo nixos-rebuild dry-build`
What is a NixOS generation?|A NixOS generation is a complete snapshot of your system configuration. Each time you run `nixos-rebuild switch`, NixOS creates a new generation. You can:\n\n- List generations: `sudo nix-env --list-generations --profile /nix/var/nix/profiles/system`\n- Rollback to previous: `sudo nixos-rebuild switch --rollback`\n- Boot older generations from the boot menu\n\nThis makes NixOS very safe - you can always go back if something breaks!
How do I search for packages in NixOS?|To search for packages:\n\n1. **Command line**: `nix search nixpkgs packagename`\n2. **Online**: Visit search.nixos.org\n3. **Show package info**: `nix-env -qa --description packagename`\n\nExample: `nix search nixpkgs firefox` shows all Firefox variants
My WiFi isn't working on NixOS|To fix WiFi on NixOS:\n\n1. **Check if interface is detected**: `ip link`\n2. **Enable NetworkManager** in /etc/nixos/configuration.nix:\n```nix\nnetworking.networkmanager.enable = true;\n```\n3. **Rebuild**: `sudo nixos-rebuild switch`\n4. **Connect**: `nmcli device wifi connect "NetworkName" password "YourPassword"`\n\nFor GUI: Install `networkmanagerapplet` package
How do I enable a service in NixOS?|To enable services in NixOS, edit /etc/nixos/configuration.nix:\n\nExample for SSH:\n```nix\nservices.openssh.enable = true;\n```\n\nExample for Docker:\n```nix\nvirtualisation.docker.enable = true;\n```\n\nThen apply: `sudo nixos-rebuild switch`\n\nServices are managed declaratively - no need for systemctl enable!
What's the difference between nix-env and configuration.nix?|Key differences:\n\n**configuration.nix** (recommended):\n- Declarative: Describes desired system state\n- Reproducible: Same config = same system\n- System-wide packages\n- Rollback entire system\n\n**nix-env**:\n- Imperative: Like apt-get or yum\n- User-specific packages\n- Not in system config\n- Harder to reproduce\n\nBest practice: Use configuration.nix for everything when possible!
How do I install VS Code on NixOS?|To install VS Code:\n\n1. **Official Microsoft version**:\n```nix\nenvironment.systemPackages = with pkgs; [\n  vscode\n];\n```\n\n2. **Open source version (VSCodium)**:\n```nix\nenvironment.systemPackages = with pkgs; [\n  vscodium\n];\n```\n\n3. **With extensions**:\n```nix\nenvironment.systemPackages = with pkgs; [\n  (vscode-with-extensions.override {\n    vscodeExtensions = with vscode-extensions; [\n      ms-python.python\n      ms-vscode.cpptools\n    ];\n  })\n];\n```
